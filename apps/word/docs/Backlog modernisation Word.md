# Backlog de modernisation - Clic Formation Word

## Objectif

Transformer l'audit de compatibilité en plan d'action concret pour rendre les supports Word pleinement exploitables avec Microsoft Word LTSC 2024 et Microsoft 365.

## Règle de priorité

- `P1` : blocant ou obsolète, à traiter avant diffusion
- `P2` : important, à traiter dans la première vague de mise à jour
- `P3` : amélioration utile, non bloquante

## Livrables attendus

Pour clôturer le backlog, produire :

- une version révisée des consignes des exercices impactés ;
- des captures d'écran Word 2024 ;
- une validation finale module par module ;
- une note de version listant les adaptations effectuées.

---

## Critères de validation

Une tâche est considérée terminée si :

- la consigne fonctionne réellement dans Word LTSC 2024 ;
- le résultat attendu est atteignable sans contournement ;
- les visuels du support correspondent à l'interface actuelle ;
- aucune référence à une fonctionnalité supprimée ne subsiste.
