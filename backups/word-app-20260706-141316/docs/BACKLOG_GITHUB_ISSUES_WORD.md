## Issue 1 - Refaire l'exercice WordArt utilisant un remplissage image supprimé

**Type**: Content update  
**Priorité**: P1

### Description

Le module `W17 - WordArt`, exercice 2, repose sur un remplissage du texte par image qui n'existe plus dans Microsoft Word LTSC 2024 / Microsoft 365.

L'exercice doit être remplacé par une variante compatible avec les versions actuelles de Word, sans perdre l'objectif pédagogique.

### Critères d'acceptation

- L'exercice ne dépend plus du remplissage d'un texte par image.
- La nouvelle consigne reste compréhensible pour un public débutant.
- Le résultat attendu est réalisable dans Word LTSC 2024.
- Les visuels d'exemple sont mis à jour.
- Le support ne mentionne plus de fonctionnalité supprimée.
